Selectors

Selectors allow you to target HTML elements in different ways. 

/* Element selector */
p {
    color: blue;
}

/* Class selector */
.container {
    padding: 20px;
}

/* ID selector */
#header {
    background-color: lightgray;
}

/* Attribute selector */
input[type="text"] {
    border: 1px solid black;
}

/* Pseudo-class selector */
a:hover {
    color: red;
}

/* Pseudo-element selector */
p::first-line {
    font-weight: bold;
}
Box Model

The Box Model describes the rectangular boxes generated for elements in the document tree. 

.box {
    width: 200px;
    padding: 10px;
    border: 5px solid black;
    margin: 20px;
}
Typography

Typography refers to the style and appearance of text. 

h1 {
    font-family: Arial, sans-serif;
    font-size: 24px;
    font-weight: bold;
    line-height: 1.5;
    text-align: center;
    text-decoration: underline;
    text-transform: uppercase;
}
Color

Color can be specified using names, hex values, RGB, RGBA, HSL, and HSLA. 

.text-color {
    color: #ff0000; /* Hex */
    background-color: rgb(0, 255, 0); /* RGB */
}

.absolute-units {
    width: 100px;
    height: 2in;
}

.relative-units {

Positioning

Positioning allows you to control the layout of elements on the page. 

.relative {
    position: relative;
    top: 10px;
    left: 20px;
}

.absolute {
    position: absolute;
    top: 50px;
    left: 100px;
}

.fixed {
    position: fixed;
    bottom: 10px;
    right: 10px;
}

.sticky {
    position: sticky;
    top: 0;
}

    display: block;

.inline {

    display: inline-block;
}

.flex {
    display: flex;

.grid {

Flexbox

Flexbox is a layout model for distributing space along a single axis. 

.flex-container {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
    justify-content: space-between;
    align-items: center;
}

.flex-item {
    order: 2;
    flex-grow: 1;
    flex-shrink: 0;
    flex-basis: 50%;
    align-self: flex-start;
}
Grid Layout

Grid layout allows you to create complex layouts with rows and columns. 

.grid-container {
    display: grid;
    grid-template-columns: 1fr 2fr;
    grid-template-rows: auto 1fr;
    grid-template-areas:
        "header header"
        "sidebar content";
    gap: 10px;
}

.header {
    grid-area: header;
}

.sidebar {
    grid-area: sidebar;
}

.content {
    grid-area: content;
}
Transitions and Animations

Transitions and animations add dynamic effects to elements. 

.box {
    transition: background-color 0.5s ease;
}

.box:hover {
    background-color: blue;
}

@keyframes slidein {
    from {
        transform: translateX(0%);
    }
    to {
        transform: translateX(100%);
    }
}

.animated-box {
    animation: slidein 3s ease-in-out infinite;
}
Transforms

Transforms allow you to change the appearance of an element using translation, rotation, scaling, and skewing. 

.transform-box {
    transform: translate(50px, 100px) rotate(45deg) scale(1.5) skew(10deg, 20deg);
}
Responsive Design

Responsive design ensures that your website adapts to various screen sizes. 

@media (max-width: 600px) {
    .responsive {
        background-color: lightblue;
    }
}
CSS Variables

CSS Variables allow you to define reusable values in CSS. 

:root {
    --main-bg-color: coral;
    --main-font-size: 16px;
}

.variable-box {
    background-color: var(--main-bg-color);
    font-size: var(--main-font-size);
}
Pseudo-classes and Pseudo-elements

Pseudo-classes and pseudo-elements help you apply styles based on the state of elements or specific parts of an element. 

a:visited {
    color: purple;
}

.first-letter::first-letter {
    font-size: 200%;
    color: red;
}
CSS Preprocessors

CSS preprocessors add features to CSS such as variables and nested rules. 

$primary-color: blue;

.preprocessor {
    color: $primary-color;
}

Best practices for organizing and optimizing CSS.
